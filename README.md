# Validation Project

## Overview
  The project implements Custom Validations for the REST API Endpoints Requests and based on the input the Validations done and output is Provided

## Technologies Used
 - JDK 17
 - Maven 3.9.6
 - Spring Boot 3.2.1
 - Jakarta Persistence API (JPA)
 - Jakarta Bean Validation
 - Swagger-ui
 - Lombok
 - Spring Data JPA
 - Spring MVC
 - MYSQL
